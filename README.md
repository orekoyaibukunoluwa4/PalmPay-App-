# PalmPay — Send Money to Other Banks: User Guide
A clear, step-by-step technical writing document that walks PalmPay users through the process of sending money from their PalmPay wallet to any bank in Nigeria.
---
## About This Document

This guide was written as a technical writing sample demonstrating how to produce clean, user-facing product documentation. It covers:

- Prerequisites before initiating a transfer
- A 12-step walkthrough with annotated screenshots
- A troubleshooting table for common errors
- Customer support contact information

The document is formatted as a professional Word file (.docx) with consistent branding, headers, footers, and page numbering.
